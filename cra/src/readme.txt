How to start this project:

1. Create your working folder;
2. Copy to this folder compressed folder 'form.rar';
3. Unpack folder 'form.rar';
4. Open console and change directory to 'cra' (it is inside compressed folder);
5. Type command: 'npm install';
6. When instaling finished type: 'npm start' to start project;

Another way is clone repository from https://github.com/jurek55/order-form
